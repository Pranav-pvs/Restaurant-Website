# 🍽️ Restaurant Website

A modern and responsive **Restaurant Landing Page** built using **HTML5** and **CSS3**. The project features a clean user interface with a hero section, feature highlights, food gallery, sticky navigation bar, and responsive design for desktop and mobile devices.

## ✨ Features

* 🍔 Modern restaurant landing page
* 📱 Fully responsive design
* 📌 Sticky navigation bar
* 🎯 Hero section with call-to-action button
* 🥗 Feature cards for restaurant services
* 🖼️ Responsive food gallery using CSS Grid
* 🎨 Clean UI with smooth hover animations
* 📧 Footer with newsletter subscription
* 🌐 Social media icons using Font Awesome
